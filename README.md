# EchoBox
EchoBox is a cloud-integrated Android music application that enables users to upload, stream, and manage their personal music libraries across multiple devices. The application leverages Firebase services to provide secure authentication, scalable storage, and real-time data synchronization.
_______________________________________________________________________________________________________________________________________________________

#Overview
EchoBox delivers a seamless music experience by combining local device interaction with cloud-based infrastructure. Users can upload audio files, organize them into playlists, and access their library from any authenticated device.
_______________________________________________________________________________________________________________________________________________________

 Key Features
	•	Secure User Authentication
	•	Firebase Authentication (Email/Password)
	•	User-specific data isolation using unique UIDs
	•	Cloud-Based Music Storage
	•	Upload and store MP3 files using Firebase Cloud Storage
	•	Efficient retrieval via secure download URLs
	•	Realtime Data Management
	•	Firestore database for storing song metadata and playlists
	•	Persistent, user-specific libraries accessible across devices
	•	Custom Music Player
	•	Play / Pause functionality
	•	Next / Previous navigation
	•	Shuffle and Repeat modes
	•	Interactive seekbar with real-time playback tracking
	•	Playlist Management
	•	Create and manage custom playlists
	•	Add and remove songs dynamically
	•	Cloud-synced playlist persistence
	•	Cross-Device Synchronization
	•	Consistent user experience across multiple devices
	•	Automatic retrieval of user-specific content upon login

Getting Started
	1.	Clone the repository
	2.	Open the project in Android Studio
	3.	Add your google-services.json file
	4.	Sync Gradle dependencies
	5.	Run the application on an emulator or physical device
